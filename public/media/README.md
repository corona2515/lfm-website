# Media Assets for LeanFM Website

This directory contains screenshots and images for the marketing website.

## Naming Convention

All images use sequential IDs:
- `IMG-001.png`
- `IMG-002.png`
- etc.

## Adding Images

1. Reference `assets-manifest.json` in the repo root
2. Find the placeholder ID you want to replace
3. Create your image matching the specified dimensions/aspect ratio
4. Save as `IMG-XXX.png` in this directory
5. The website will automatically use the real image

## Image Guidelines

- Use PNG format for screenshots
- Ensure text is readable at display size
- Match the visual style of the actual OnPoint application
- Remove any sensitive/test data from screenshots
- Optimize file size while maintaining quality

## Screenshot Tips

For UI screenshots:
- Use a clean, representative state
- Show realistic but anonymized data
- Ensure consistent window size
- Capture at 2x resolution if possible for retina displays
